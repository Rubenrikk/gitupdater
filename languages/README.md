# Languages Folder

This folder contains translation files for the Auto-Deploy for GitHub plugin.

## Translation Files

- `.pot` - Template file for translators
- `.po` - Translation files for each language
- `.mo` - Compiled translation files

## Supported Languages

Currently the plugin is available in:
- English (default)
- Dutch (Nederlands)

## Adding Translations

To add a new translation:
1. Create a `.po` file for your language
2. Compile it to a `.mo` file
3. Place both files in this folder

## WordPress.org Compliance

This folder is required for WordPress.org plugin submission compliance.
